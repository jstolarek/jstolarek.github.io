Fremen Zone: Creating a Website Using Only HTML and CSS

Last July I wrote a post about all the websites that I have run in my life. An important part of that list was Fremen Zone, a fansite dedicated to Frank Herbert’s “Dune” that I created back in high school and have recently restored from backups on my hard drive. I ended the post with an idea of re-doing Fremen Zone from scratch:

Restoring the Fremen Zone back to life sparked a new desire in me. While I said I have no intention of updating it, I feel like re-doing the whole site from scratch. (…) And so I am slowly gathering material for a new Dune fanpage. All the lessons above have taught me to avoid complexity and stay as close to the basics as possible. And so my plan is to create a new website based on pure HTML and CSS.

Thus for the past couple of months I have been hard at work to create a new version of Fremen Zone. It finally went live on Febuary 1st 2024 and is available at https://www.fremenzone.pl. In this post I want to focus solely on technical aspects of the page, in particular writing a simple static page generator.
